raafat-elhagan
==============

Rafaat Elhagaan is a First Person Shooter game implemented in C++ using OpenGL for academic purpose.

### Depenedencies
#### For Linux
```
sudo apt-get install mpg321
```

### How to run it
#### For Linux
```
make
./main.bin

```

#### For Mac
```
make mac
./main.bin
```
